 # FAQ Chatbot using NLP

## Project Overview
This project is a simple FAQ Chatbot developed using Python and Natural Language Processing (NLP) techniques. The chatbot matches user queries with predefined Frequently Asked Questions (FAQs) and returns the most relevant answer.

## Features
- Interactive question-answer chatbot
- FAQ-based response system
- Text vectorization using TF-IDF
- Similarity matching using Cosine Similarity
- Fast and efficient answer retrieval

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## How It Works
1. FAQ questions and answers are stored in a dataset.
2. User input is converted into numerical vectors using TF-IDF.
3. Cosine Similarity is used to compare the user's query with existing FAQ questions.
4. The chatbot identifies the most similar question and displays the corresponding answer.

## Sample Questions
- What is Python?
- What is Machine Learning?
- What is NLP?
- What is GitHub?
