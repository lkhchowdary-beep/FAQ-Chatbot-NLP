FAQ Chatbot using NLP
Project Overview
This project is a simple FAQ Chatbot developed using Python and Natural Language Processing (NLP) techniques. The chatbot matches user queries with predefined Frequently Asked Questions (FAQs) and returns the most relevant answer.
Features
Interactive question-answer chatbot
FAQ-based response system
Text vectorization using TF-IDF
Similarity matching using Cosine Similarity
Fast and efficient answer retrieval
Technologies Used
Python
Scikit-learn
TF-IDF Vectorizer
Cosine Similarity
How It Works
FAQ questions and answers are stored in a dataset.
User input is converted into numerical vectors using TF-IDF.
Cosine Similarity is used to compare the user's query with existing FAQ questions.
The chatbot identifies the most similar question and displays the corresponding answer.
